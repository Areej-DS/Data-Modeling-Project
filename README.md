# 👗 Fashion MNIST Classification

A deep learning project that classifies fashion items into 10 categories using the Fashion MNIST dataset.

## 📌 Overview
Comparing multiple machine learning and deep learning models to classify fashion items such as T-shirts, trousers, dresses, shoes, and bags.

## 🧠 Models Used
- CNN (Convolutional Neural Network)
- FNN (Feedforward Neural Network)
- Random Forest
- KNN (K-Nearest Neighbors)
- SVM (Support Vector Machine)

## 👗 Categories
T-shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle Boot

## 🛠️ Built With
- Python
- TensorFlow / Keras
- Scikit-learn
- Matplotlib
- NumPy
